## [C++ Zlib 库](#)

**Zlib** 是一个通用的数据压缩库，所有代码都是线程安全。最早是由 `Jean-loup Gailly` 与 `Mark Adler` 所开发。今天，zlib是一种事实上的业界标准，以至于在标准文档中，`zlib` 和 `DEFLATE` 常常互换使用, 数以万计的应用程序直接或间接依靠zlib压缩函式库，包括： `Linux核心`、`libpng`、`Apache`、`nginx`等等。
